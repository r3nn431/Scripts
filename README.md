# Scripts

## 1: Nexus Mods - Hide Translations from Requirements
A simple userscript for Violentmonkey / Tampermonkey that cleans up the "Requirements" tab on Nexus Mods by hiding any mods that are already listed as translations.
The script scans the Translations table for mod IDs and then checks the Requirements table. If a match is found, the requirement row is automatically hidden.

Installation: https://raw.githubusercontent.com/r3nn431/Scripts/main/nexus-hide-translations.user.js
